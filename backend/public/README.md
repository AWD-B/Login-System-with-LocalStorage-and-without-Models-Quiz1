# Pet Care Guides

This directory contains downloadable PDF guides for pet care.

## Required PDF Files:

1. `dog-training-guide.pdf` - Complete Dog Training Guide
2. `cat-grooming-guide.pdf` - Cat Grooming Mastery  
3. `bird-care-guide.pdf` - Bird Care Essentials
4. `fish-tank-guide.pdf` - Fish Tank Maintenance
5. `rabbit-nutrition-guide.pdf` - Rabbit Nutrition Guide
6. `small-pet-housing.pdf` - Small Pet Housing Guide

## Adding New Guides:

1. Place PDF files in this directory
2. Update the guides array in `routes/serviceRoutes.js`
3. Ensure file permissions allow public access
